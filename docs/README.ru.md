# RZD Tickets MCP

Основной README на русском находится здесь: [../README.md](../README.md).

Коротко: это read-only MCP-сервер для агентов, которым нужно видеть наличие
билетов, вагонов, нижних/верхних мест, соседних пар, фото вагонов при наличии
в РЖД и ссылок РЖД для ручного оформления.

Классы обслуживания РЖД не моделируются как enum. Сервер возвращает сырой код,
человекочитаемое описание, официальный transcript и осторожные подсказки,
чтобы агент понимал смысл без постоянного расширения локального списка кодов.

Фото вагонов находятся в `imageInfo`. Если РЖД вернул `HasImages=false`, массив
`images` будет пустым, а причина будет указана в `unavailableReason`.
